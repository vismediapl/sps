Introduction

eStats is a script that allows to manage complex website statistics, it collects very many different informations about users and theirs activity.

Features

Script allows to:

  • collect and display information about used OSes, browsers, robots, hosts, keywords, referrers, screen resolutions, websites popularity, flash plugin version, etc.;
  • configure the script by the control panel with password protection;
  • exclude counting visits from choosen IP adresses;
  • protect preview of stats by password;
  • easy change of look thanks to themes;
  • collect and display informations about visits in the last twenty - four hours, month and year, in addition stats archive;
  • displays detailed informations about specified amount of last visitors.

Requirements

Required PHP version is 4.3.0 or higher, it works without problems on PHP 5.x.

Support on the server at least one of the following databases is required:
SQLite (version 2.8), MySQL (mysqli extension or mysql, require adequately minimum database version 4.1.3 and 3.0.23) or PostgreSQL.

Installation

  1. Copy whole estats directory into Your server;
  2. Set up CHMOD 777 for data/ directory and CHMOD 666 for file conf/config.php (create them if do not exist);
  3. Open in the browser directory where You copied files;
  4. Follow displayed instructions;
  5. You must put into sources of monitored websites this code (script will work correctly only if the server alllows to execute PHP code in the website file!): 
    <?php
define ('eCOUNT', 1);
define ('eADDRESS', 'ADDRESS');
define ('eTITLE', 'TITLE');
@include ('PATH');
?>
    Where:

    TITLE - if file text encoding is different than UTF-8, You shouldn't use characters outside English alphabet;
    ADDRESS - website address (without http://website.address.com, example: /index.php?page=main);
    PATH - path to file stats.php in estats directory, example: ../directory/estats/stats.php.

    If You will use this code then all omitted parameters will be generated automaticlly (website title will be set to it's address): 
    <?php
define ('eCOUNT', 1);
@include ('PATH');
?>
  6. Collecting informations about screen resolutions, flash plugin version etc. will work only if You put this code into the website source: 
    <noscript>
<a href="http://estats.emdek.cba.pl/">
<img src="PATHstats.php?estats=1" alt="eStats" title="eStats" />
</a>
</noscript>
<script type="text/javascript">
var ePath = 'PATH';
var eCount = COUNT;
</script>
<script type="text/javascript" src="PATHstats.js"></script>
    Where:

    PATH - path to estats directory, example: /directory/estats/,
    COUNT - 0 - collect only technical data, 1 - collect all data (only if You could not use PHP code).
